ເຮັດຄວາມຮູ້ຈັກກັບ Python


ພາສາໄພທອນ (python) ເປັນພາສາໜຶ່ງ ທີ່ຖືກນຳມາໃຊ້ງານຢ່າງກວ້າງຂວາງ ເນື່ອງຈາກເປັນພາສາທີ່ງ່າຍ ສາມາດຮຽນຮູ້
ແລະ ເຂົ້າໃຈໄດ້ຢ່າງວ່ອງໄວ. ນອກຈາກນີ້ Python ຍັງສາມາດປະຍຸກໃຊ້ງານກັບຫຼາກຫຼາຍວຽກງານ ແລະ ຮອງຮັບການເຮັດ
ວຽກເທິງອຸປະກອນຫຼາຍໆຮູບແບບໄດ້. ເຮັດໃຫ້ Python ເປັນພາສາໂປຣແກຣມທີ່ໄດ້ຮັບຄວາມນິຍົມຫຼາຍທີ່ສຸດພາສາໜຶ່ງ ໃນປັດຈຸບັນ.

#ແນະນຳພາສາ Python


Python ເປັນພາສາໂປຣແກຣມທີ່ພັດທະນາຂຶ້ນໂດຍ Guido van Rossum ແລະ ທີມພັດທະນາເຊິ່ງມີຈຸດປະສົງເພື່ອຕ້ອງການຢາກ
ພັດທະນາພາສາ ທີ່ໃຊ້ສ້າງໂປຣແກຣມດ້ວຍໂຄງສ້າງທີ່ງ່າຍຂຶ້ນ ແລະ ເປັນພາສາແບບເປີດເຜີຍລະຫັດ (Open Source) ທີ່ມີ
ການນຳໄປພັດທະນາຕໍ່ໆກັນເລື້ອຍໆເຮັດໃຫ້ Python ມີຄວາມສາມາດຫຼາຍຂຶ້ນ, ມີຄຸນລັກສະນະທີ່ຫຼາກຫຼາຍ ສາມາດນຳໄປໃຊ້ພັດ
ທະນາວຽກງານດ້ານຕ່າງໆໄດ້ ເຊັ່ນ: ວຽກງານດ້ານວິທະຍສາດ, ດ້ານຖານຂໍ້ມູນ, ປັນຍາປະດິດ ແລະ ສະໜັບສະໜູນການເຮັດ
ວຽກໄດ້ທຸກແພລັດຟອມ ໂດຍມີຄຸນສົບັດທີ່ໂດດເດັ່ນດັ່ງນີ້:

▸ງ່າຍຕໍ່ການຮຽນຮູ້ ແລະ ການທຳຄວາມເຂົ້າໃຈ ເນື່ອງຈາກໂຄງສ້າງພາສາບໍ່ມີຄວາມຊັບຊ້ອນ

▸ມີການຈັດການໜ່ວຍຄວາມຈຳແບບອັດຕະໂນມັດ ສາມາດຈັດການພື້ນທີ່ໃນໜ່ວຍຄວາມຈຳໄດ້ຢ່າງມີປະສິດທິພາບ

▸ເຮັດວຽກໄດ້ໃນທຸກລະບົບຄື ມີຄຸນລັກສະນະຂອງ Platform Independent

ການເຮັດວຽກຂອງໂປຣແກຣມພາສາ Python


ພາສາ Python ສາມາດເຮັດວຽກຮ່ວມກັບຕົວແປພາສາແບບອິນເຕີພລີເຕີ (Interpreter) ໂດຍເຮົາສາມາດພິມຄຳສັ່ງເປັນໃນລັກ
ສະນະຂອງພາສາສະຄຣິບ ໃຫ້ຄອມພິວເຕີເຮັດວຽກໂຕ້ຕອບເທື່ອລະຄຳສັ່ງ ແລະ ສະແດງຜົນແຈ້ງເຕືອນທັນທີຫາກໂປຣແກຣມເກີດມີຂໍ້ຜິດພາດ.
ນອກຈາກນັ້ນເຮົາຍັງສາມາດຂຽນໂປຣແກຣມລຽງຕໍ່ກັນເປັນໄຟຣ໌ໄດ ໂດຍໄຟຣ໌ Source Code ຈະມີນາມສະກຸນເປນ .py ເມື່ອມີການຣັນ
ໂປຣແກຣມ Source Code ດັ່ງກ່າວຈະຖືກນຳໄປສ້າງເປັນ Python Byte Code ທີ່ນຳໄປໃຊ້ງານໄດ້ເທິງຫຼາຍແພລັດຟອມ.
Python byte Code ຈະຸືກຄອມໄພລ໌ດ້ວຍ Python Virtual Machine (PVM) ໃຫ້ເປັນພາສາສະເພາະຂອງແພລັດຟອມນັ້ນ ເພື່ອໃຫ້
ໂປຣແກຣມສາມາດເຮັດວຽກໄດ້.

ແນະນຳເຄື່ອງມືສຳລັບຂຽນໂປຣແກຣມພາສາ Python


ໃນການພັດທະນາໂປຣແກຣມດ້ວຍພາສາ Python ຈະມີ່ເຄືອງມືທີ່ຮອງຮັບການພັດທະນາທີ່ຫຼາກຫຼາຍ ເຊິ່ງເຄື່ອງມືແຕ່ລະຕົວກໍ່ມີຄຸນສົມບັດ
ທີ່ໂດດເດັ່ນແຕກຕ່າງກັນໄປເຊັ່ນ:

▸Komodo IDE ເປັນຊຸດເຄື່ອງມືທີ່ມີຄ່າລິຂະສິດ (ເສຍຄ່າໃຊ້ຈ່າຍໃນການໃຊ້ງານ) ຮອງຮັບໄດ້ຫຼາຍພາສາເຊັ່ນ: Python, PHP, C, JavaScript ເປັນຕົ້ນ.

▸PyScripter ເປັນຊຸດເຄື່ອງມືສຳລັບພັດທະນາພາສາ Python ເທິງລະບົບປະຕິບັດການ Windows ທີ່ສາມາດໃຊ້ງານໄດ້ຟຣີ.

▸ NetBeans ເປັນເຄື່ອງມືທີ່ໃຊ້ສຳລັບພັດທະນາແອັບພລິເຄຊັ່ນດ້ວຍພາສາຕ່າງໆ ແລະ ໜຶ່ງໃນນັ້ນກໍ່ຄືພາສາ Python ໂດຍໄດ້ຮັບການສະໜັບສະ


ໜູນຈາກບໍລິສັດ Sun Microsystems ສາມາດໃຊ້ງານໄດ້ຟຣີ.

▸Python IDLE ເປັນເຄື່ອງມືທີ່ລວມຢູ່ໃນຊຸດອິນເຕີພລີຂອງໂປຣແກຣມພາສາ Python ຢູ່ແລ້ວ.

▸PyCharm Education ເປັນຊຸດເຄື່ອງມືຂອງບໍລິສັດ JetBrains ເຊິ່ງເປັນບໍລິສັດທີ່ພັດທະນາເຄື່ອງມືສຳລັບຂຽນໂປຣແກຣມ ໂດຍໂປຣແກຣມ

PyCharm ເວີຊັ່ນ Education ສາມາດໃຊ້ງານໄດ້ຟຣີ
